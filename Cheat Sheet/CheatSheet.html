Word Wide Web General

1.What are the most essential technologies that made the World Wide Web such success?

The early development of the world wide web was based on providing a basic public good for sharing knowledge. That was key to its success.

2.How are resources (html pages,images,etc) addressed in the web?

The browser sends an HTTP request message to the server, asking it to send a copy of the website to the client (you go to the shop and order your goods).

Web Server

1.What is a web server?

A web server is a computer that runs websites.

2.What does a web server?

The basic objective of the web server is to store, process and deliver web pages to the users.

3.How can content be stored on a web server?

They store data on the server using some kind of database(server-side storage), then run server-side code to retrieve needed data.

4.How is content in webpages descriped?

Web content refers to the textual, aural, or visual content published on a website.

Web Client

1.What is a web client?

A Web client typically refers to the Web browser in the user's machine or mobile device.

2.What does a web client?

A web client is a client-side application used for connecting to a web server over HTTP. It is typically a web browser or web app which displays web pages received from the server and allows users to interact with the web server.

3.Web Client and Web Browser is the same

Client software is a specific application that's installed on your computer, that can be used to communicate to some other piece of software over a network (such as the internet's world wide web).

Hello World

1.What is the html head element?

The <head> element is a container for metadata (data about data) and is placed between the <html> tag and the <body> tag.

2.What does <!DOCTYPE html> mean?

A "Document Type Declaration" is an SGML concept for signalling the mark-up syntax and vocabulary for the mark-up that follows.

3.What are the two top most elements within the html element?

An HTML 4 document is composed of three parts: a line containing HTML version information, a declarative header section (delimited by the HEAD element), a body, which contains the document's actual content.

4.How to emphasize a text and how is it displayed in the browser?

The HTML <em> tag marks text that has stress emphasis which traditionally means that the text is displayed in italics by the browser.

5.How to mark up important text and how is it displayed in the browser?

<b> - Bold text. <strong> - Important text. <i> - Italic text. <em> - Emphasized text. <mark> - Marked text. <small> - Smaller text. <del> - Deleted text. <ins> - Inserted text.
